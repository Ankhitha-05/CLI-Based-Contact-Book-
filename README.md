# 📒 CLI-Based Contact Book

A simple Command Line Interface (CLI) based Contact Book application built using Python. This project allows users to manage contacts efficiently through the terminal.
## 🚀Features

* ➕ Add new contacts
* 📋 View all contacts
* 🔍 Search contacts by name
* ❌ Delete contacts
* 💾 Data stored using Python lists and dictionaries
